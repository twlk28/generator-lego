# generator-lego

> f2e workflow, base on yeoman & gulp

## Getting Started
1. `npm install -g yeoman`
2. `npm install -g gulp`
3. `npm install -g generator-lego`
4. `yo lego`
5. `gulp`

## Usage

### `gulp`
创建一个链接，自动检测`src`文件夹下的静态文件，自动刷新。支持sass编译。

### `gulp -p 8080`
同上，`-p`参数指定特定端口。

### `gulp release`
将静态文件压缩到`dest`文件夹下。
